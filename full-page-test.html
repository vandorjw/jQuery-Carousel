Feature List:

The Carousel is still lacking many features and still has many bugs. A list of items I plan to implement are:

  • Zoom image and pause on click
  • Browser width detection
  • Proper pagination
  • Validate on jslint
  • Pause/Start
  • Fix flash of Content during intial load
  • Touch events, swiping left and right
  • Allow less than 3 images
  • Smooth the transition

All styling is done using css. The only "style" applied via javascript, is changing the display from "none" to "inline". The things that matter, is the image 'alt' and the the class names. Both of these are for-loop friendly.
